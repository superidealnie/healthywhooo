# HealthyWhooo Project Scope

## Problem
People and pet owners struggle to understand complex ingredient lists on food packaging.

## Solution
HealthyWhooo scans ingredient lists and translates them into simple explanations.

Users can check ingredients for:
- humans
- dogs
- cats

## Key features
- ingredient scanning
- ingredient search
- simple explanations
- save ingredient
- pet ingredient analysis

## Out of scope
- full nutrition tracking
- supplement recommendations
- user authentication
