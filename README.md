# HealthyWhooo

HealthyWhooo is a mobile-first ingredient education app that helps users understand food ingredients in simple language.

Users can:
- scan ingredient lists
- search ingredients manually
- get simple explanations
- check products for humans, dogs, and cats

## Project Goal
Help people and pet owners make more confident food choices.

## Current MVP Scope
- Guide selection
- Ingredient search
- Sample ingredient lists
- Ingredient explanation cards
- Save ingredient
- Coming Soon: Nutrient Scanner
